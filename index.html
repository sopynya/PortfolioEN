DinoDatabase

DinoDatabase is an informative and responsive website focused on scientific dissemination about dinosaurs.
Developed with HTML and CSS and emphasizing user experience, the project features individual pages for different species, with organized content, attractive visuals, and friendly URLs.
It uses good SEO practices, responsive design, and modern styling with custom fonts and soft gradients. The goal is to make knowledge about paleontology accessible and engaging for diverse audiences.

Great Recipes

This project uses React and Vite, focusing on an interactive and intuitive experience. It allows users to explore and filter recipes from different regions of the world, with search options by type (vegan, non-vegan, drinks) and flavor (sweet, savory).
Using React Router for dynamic navigation, this project offers a responsive and smooth interface, with illustrative images for each recipe. It’s a great demonstration of how to integrate search and filtering functionalities in a SPA (Single Page Application).
