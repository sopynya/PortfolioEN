About me

Hello! My name is Sophia, and I have a special passion for programming, dinosaurs, and cats.
I have hands-on experience with HTML, CSS, JavaScript, Python, Git, VSCode, React.js, Next.js, PostgreSQL, MySQL, MongoDB, BaaS (ex. SupaBase, AppWrite), Zustand, TanStack NestJS and express.js, which enables me to develop well-structured, functional, and optimized digital solutions for the web.

Dino API

A dinosaur API developed with Next.js, designed as a scientific database with organized information about species, geological periods, and key characteristics. The project explores API routes, the modern Next.js structure, and data organization to facilitate queries and educational use.

DinoDatabase

DinoDatabase is an informative and responsive website focused on the scientific outreach of dinosaurs. Developed using React.js, CSS, and JSON, with a strong focus on user experience, the project presents a wide list of dinosaurs and their information, featuring well-organized content, an attractive visual design, and user-friendly URLs.
The goal is to make knowledge about paleontology accessible and engaging for a wide range of audiences.

Frontpage

Front Page is a customizable content aggregator that pulls RSS and Atom feeds into a single, well-designed reading dashboard. It allows users to organize multiple sources in one place, track new articles, and create a personalized news experience.
The project was created using NextJs, Zustand for global state, Tanstack Query, RSS parser and a JSON file to populate the guest feed.

Fishis

Fishis is a fishing game developed with Next.js where the player collects fish, sells their catches, and upgrades their equipment by equipping rods and hooks to improve performance. The project uses TanStack Query for asynchronous data management, Zustand for global state, PostgreSQL as the database (hosted on Neon), and JWT-based authentication. The application was built with a focus on organization, scalability, and modern full-stack architecture.
